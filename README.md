rat-vec
=======
Exact rational vector arithmetic.

<img src="img/rat.jpg">

# Example

```javascript
var vec = require('rat-vec')

var toFloat = require('rat-vec/to-float')

var add = require('rat-vec/add')
```

# Install

```
npm i rat-vec
```
